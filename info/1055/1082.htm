New Hardware Driven Storage and Computing Systems

APPT

As the emerging of various new hardware devices for storage, computing and networking systems, like NVRAM, GPU, RDMA, etc., the storage and computing system design faces a lot of new opportunities and challenges. Research on new hardware driven storage and computing system has also received a lot of attentions recently, e.g., flash or NVRAM aware file systems, GPU boosted graph and big data analytic systems, and RDMA driven distributed systems, etc, while there are still lots of critical issues to be addressed to fully take advantage of these new hardware devices. This forum is to bring together the researchers and experts working in this area to share their recent works and discuss/explore future directions and topics in this area.

The major topics include:

Ø Flash/NVRAM aware storage and file system: storage architecture, tiered storage, index, RAID, erasure coding, etc.

Ø Distributed systems with high-speed networks: parallel and distributed file systems, in-memory systems, disaggregated storage, cloud storage systems, etc.

Ø New hardware accelerated computing/applications: graph systems, ML systems, big data analysis, etc.

Ø New storage APIs: object storage, key-value stores and others.

Ø Data protection and recovery, data compression and deduplication, data preservation and management.

Ø Etc.

王肇国 上海交通大学


报告及嘉宾简介：

报告1：

嘉宾简介：陈康，男，1976年生于浙江。清华大学计算机科学与技术系研究员。研究领域为面向大数据的高性能计算系统，在高可扩展高可靠数据存储技术，数据在不同计算节点和计算域之间快速流动技术，高性能图数据计算系统的设计与优化等方面作出了一系列创新性的成果。受邀担任计算机系统领域顶级和重要国际会议SOSP，OSDI、EuroSys、Middleware、ICDCS、IPDPS等的程序委员会委员。研究的成果受到广泛的重视和应用。安全区域的单一地址空间Occlum已经用于蚂蚁金服的安全计算产品SOFAEnclave；云存储的数据共享机制广泛应用于众多企事业单位；基于大规模纠删码的高可靠存储系统由同方公司进行产业化推广；图模型上的高效能数据计算方法已经实际应用于腾讯的图计算系统Plato。近年来发表在CCF A类期刊如IEEE TPDS、IEEE ToC，ACM ToS以及顶级与重要国际会议（OSDI / SOSP / ASPLOS / ATC / HPCA / VLDB / EuroSYS / ATC / ICS / HPDC）论文20余篇。实现了国内高校师生OSDI论文零的突破。获国家技术发明奖二等奖1项，省部级特等奖1项、一等奖2项。

报告题目：非易失性内存索引的考虑因素

报告摘要：随着商用非易失性内存的出现（Intel Optane DC PMM），是时候将之前工作提出的持久化数据结构应用到实际系统中。本文主要讨论了在将持久化数据结构应用到实际系统中需要考虑的一些因素，其中包括功能，性能和正确性。我们基于自适应基数树设计了一个新的持久化索引：范围查询优化的自适应基数树（ROART），并且考虑了上述所有的设计因素。我们分别提出了多种优化，（1）叶节点压缩方法来优化范围查询这一功能，（2）三种优化（条目压缩，元数据选择性持久化，最小指令序分裂操作）来最小化持久化开销来提高性能，（3）快速持久化内存管理方案来防止内存泄漏，并且提出了瞬间重启的策略来消除原本较长时间的恢复开销。根据测试表明，本文提出的范围查询优化的自适应基数树相比目前最新的基数树性能高至多1.65倍，并且比最新的持久化B+树高1.17～8.27倍。


报告2：

嘉宾简介：陆游游，清华大学计算机系副教授、特别研究员、博士生导师。主要研究方向是非易失性存储、文件系统及分布式存储等，在FAST、OSDI、USENIX ATC、ASPLOS等国际顶级会议期刊上发表论文30余篇，获得IEEE NVMSA 2014大会最佳论文奖和MSST 2015最佳论文提名奖。曾获中国计算机学会优秀博士学位论文奖、ACM中国操作系统分会新星奖、教育部技术发明一等奖、中国计算机学会技术发明一等奖等，入选首届中国科协青年人才托举工程、国家自然科学基金优青资助。

报告题目：存储协议与新型硬件

报告摘要：新型存储硬件与RDMA硬件访问技术对存储系统的设计带来了诸多机遇和挑战。本报告将讨论：新硬件对传统存储协议有什么影响？新硬件上传统存储协议是否有新的设计？报告将主要从存储一致性、缓存一致性、NUMA访问等存储协议探讨新硬件上的存储协议设计。


报告3：

嘉宾简介：杜子东，博士，现为中国科学院计算技术研究所副研究员，长期从事人工智能体系结构研究。他在深度学习处理器方向做出了一系列开创性的工作，以第一作者身份发表多篇国际顶级会议或期刊论文，曾获得了 CCF A 类会议 ASPLOS’14 最佳论文奖（亚洲首次在计算机体系结构领域顶级国际会议首次获奖），在 ISCA’16 上中获评审最高分，入选 IEEE Micro 评选的国际计算机体系结构领域年度十佳论文。他先后获得多个奖项，包括中科院杰出成就奖、中科院院长特别奖、中科院优秀博士论文奖和中国计算机学会CCF优秀博士学位论文奖。

报告题目：分形冯诺依曼机器学习计算机

报告摘要：机器学习是目前应用广泛的一系列人工智能算法，相应的，不同规模的机器学习计算机也被广泛的应用到不同的设备中，小至IoT设备，大到数据中心的服务器集群。目前，几乎所有的机器学习计算机的研究都局限于性能提升和能耗降低。然而，随着硬件效率特别是专用机器学习处理器效率的飞速提升，编程效率，已经取代性能和能耗成为了最致命的瓶颈，制约着机器学习计算机的广泛应用。针对在广泛众多深度学习处理器上的编程效率低的问题，我们提出了分形冯·诺依曼深度学习处理器体系结构，其中每个分形处理器（整体）由多个子分形处理器（局部）构成，整体和局部结构、指令和接口相同（分形思想），从而将*N*个深度学习处理器上的编程开发问题变成*1*个深度学习处理器上的编程开发问题。分形冯·诺依曼深度学习处理器体系结构使一个程序可以运行在不同规模的分形深度学习处理器上，从而将各种智能超算、智能服务器、智能手机，乃至智能物体的智能处理体系结构统一起来，让不同设备具有同构的、统一的软硬件，可以共享同一套开发工具、同一套软件栈，运行同一套程序，从而极大地提高了编程效率和机器学习计算机的处理效率。


报告4：

嘉宾简介：周健，华中科技大学武汉光电国家研究中心副教授。2016年获得华中科技大学计算机科学博士学位，2018年获得美国中佛罗里达大学计算机工程博士学位。2018-2020年，进入美国中佛罗里达大学做博士后。2021年，获聘为华中科技大学武汉光电国家研究中心副教授。主要从事计算机系统结构、存储级内存、可信计算、固态存储技术、大规模存储系统、大数据分析系统、近数据处理等方面的研究工作。在ISCA、DAC、IEEE TC、IEEE TPDS、ACM ToS、EuroSYS、IPDPS、DATE、ICCD、JPDC、CLUSTER等国际一流学术会议及期刊发表论文20余篇。获得发明专利3项。参与美国国家自然科学基金项目5项。参与973、863、国基金重点项目和国基金项目多项。担任IEEE TC、IEEE TPDS和IEEE TCC等计算机顶级期刊的审稿人。在IEEE CyberSciTech、IEEE DASC、IEEE PICOM、IEEE DATACOM等会议中担任过分会主席、Publication Chair、Registration Chair、Website Chair和审稿人。

报告题目：加密内存的细粒度写时复制技术

报告摘要：现代操作系统在内存管理中大量使用了写时复制之类的批量内存操作。写时复制技术大大节省了内存的空间开销、并提高了应用程序的性能。采用写时复制技术的操作系统对已分配的虚拟页面不会立即写入，而是将它们分配给共享的物理页面，从而消除大量复制或初始化开销。除了改善内存的空间效率外，写时复制技术还通过禁止写入共享页面，在首次写入时将页面内容复制到新的物理页面上，因此可以带来明显的性能和空间优势。 然而，由于需要复制或初始化新页面，对每个页面首次写访问的延迟将会增加。特别是在使用写入带宽受限的新型非易失性存储器件时，该延时将会更加严重。与此同时，为了适应未来的大容量内存，现代操作系统和处理器都增加了对大页映射的支持。通过以2MB、1GB或者更大粒度管理内存映射，可以有效减少内存映射的查询开销。然而，采用大页映射的同时，写时复制带来的延迟将会显著增加。为解决该问题，我们需要启用细粒度写时复制，仅复制页面内的已更新缓存行。通过使用细粒度的批量内存操作，并延迟其实际写入到缓存行被真正更新或复制，可以避免大量操作系统中的内存复制操作开销，从而显着提高应用程序的性能。然而，细粒度的写时复制管理会增加大量的元数据和相关地址转换开销。为此，本文提出了Lelantus，一种利用安全内存元数据来实现细粒度的写时复制操作的新方法。由于非易失性内存掉电后数据不丢失，为了保证数据安全，非易失性内存一般与加密内存技术配合使用。Lelantus依靠一种新颖的软硬件协同设计，利用加密内存的元数据跟踪被复制页面的已更新缓存行，从而将其余缓存行的复制推迟到第一次写入为止。在4KB页大小下，Lelantus可最大提升性能达3-3.5倍。当Lelantus被部署在大页面（例如2MB或1GB）上时，性能提升将变得更为明显，最大达到20倍以上。


报告5：

嘉宾简介：郑龙，华中科技大学副教授，主要研究方向包括大数据处理体系结构、运行时系统、并行编程语言等，在MICRO、FPGA、ATC、TCAD、TPDS等重要国际会议和期刊上发表论文近40篇，曾获第七届CCF大数据学术会议最佳学术论文奖，PACT 2018最佳论文提名、CGO 2015最佳报告奖，他是CCF Transactions on High Performance Computing等知名期刊的客座编辑，承担近10项国家重点研发计划项目、重点基金、面上基金等项目的研发工作。

报告题目：面向复杂图计算场景的片上缓存设计

报告摘要：作为表达和处理关联关系的典型模式，图计算成为业界的关注重点和研究热点，硬件加速是提升图计算效率的重要技术途径，在常规计算场景中(例如BFS，PageRank，SSSP等)取得显著进展。然而，以动态图、图匹配为代表的现实图计算场景多呈现结构动态多变、访存高度随机等复杂特征，对图计算的架构设计带来了严峻挑战。围绕动态图计算、图匹配分析等典型应用，报告将系统性地分析复杂计算场景的访存特征，并探讨复杂图计算片上缓存架构设计的一些探索。


报告6：

嘉宾简介：王肇国，上海交通大学电子信息与电气工程学院副教授，博士生导师。此前，他曾在纽约大学担任研究助理教授。他的研究兴趣包括分布式系统，数据库和形式化验证等。相关成果发表在OSDI, SIGMOD, NSDI, PODC, PPoPP, EuroSys 和Usenix ATC等国际重要会议上。他曾受邀担任系统软件相关领域国际著名学术会议IEEE ICDCS 2020程序委员会委员，TPDS 2020/2021专刊程序委员会委员，汇刊CACM 2017，TPDS 2019审稿人，ACM APSys 2018宣传主席，ACM SIGOPS ChinaSys 2021 大会主席。

报告题目：对学习型存储系统的思考和探究

报告摘要：如何利用机器学习技术优化存储系统是近年来的热点问题。本次报告将从我们的近期工作出发，介绍我们探究该问题的心路历程。首先将介绍我们对学习型索引结构(Learned Index）的研究，解决了已有学习型索引结构并行能力差的问题（PPoPP’20）。然后将介绍我们对学习型并发控制协议的研究（Learned Concurrency Control)，利用机器学习技术自动合成并发控制协议，最大化数据访问和处理效率（OSDI’21）。最后将总结使用机器学习优化存储系统的经验教训，并简单汇报我们在研究过程中得到的启发。
